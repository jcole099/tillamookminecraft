To access the Tillamook Minecraft server:

  1. Email your Java Minecraft username to admin@tillamookminecraft.com to be granted access to our server. *This may take up to 24 hours (Please check your junk/spam folder for a confirmation/reply email).
  2. Open and log into the game.
  3. Select "Multiplayer".
  4. The first time you connect click "Add Server".
  5. Under "Server Name" enter any name you like.
  6. Under "Server Address" enter: mc.tillamookminecraft.com and then click "Done".
  7. Select the server by the Server Name you just entered and click "Join Server".

You only need to take these steps once. After you've added the server it will be there each time you log into "multiplayer".

Tillamook Minecraft's Server Rules

Tillamook Minecraft (TM) is an all inclusive server. Although we hope to create a friendly online place where locals of Tillamook, Oregon can play together, we welcome everyone. TM is a family-friendly server that uses a whitelisting system to ensure the fun and safety of everyone on the server.

  • Be respectful of other players.
  • Don't destroy other people's work.
  • Don't steal other people's items.
  • Do not use foul/offensive language.
  • PvP is disabled.
  • Cheats/hacks are not allowed (Xray, flyhack, speedhack, etc).
For more information, please contact the administration team at admin@tillamookminecraft.com.

Server Commands

These commands can be typed into chat while in game.

Land Claims

Command Purpose
/permissiontrust <playername> Grants a player permission to share their permission level with others.
/trust <playername> Gives another player permission to build in the player's claim.
/containertrust <playername> Gives a player permission to use the player's buttons, levers, beds, crafting gear, containers, and animals.
/accesstrust <playername> Gives a player permission to use the player's buttons, levers, and beds.
/givepet <playername> Gives away a tamed animal.
/untrust <playername> Revokes any permissions granted to a player in the player's claim.
/untrust All Removes all permissions for all players in the player's claim.
/subdivideclaims Switches the player's shovel to subdivision mode, so the player can subdivide their claims.
/basicclaims Puts the player's shovel back in basic claims mode.
/abandonclaim Deletes the claim the player is standing in.
/abandonallclaims Deletes all of the player's claims.
/claimslist Displays a list of the player's claims.

Teleporting

In this section, the word "home" is synonymous with "teleport location/destination".

Command Purpose
/homes Displays a list of homes.
/home <name> Teleports the player to the specified home.
/sethome <name> Creates a new home.
/delhome <name> Removes a home.
/spawn Teleports the player to the map's spawn point.
/back Teleports the player to the previous location.
/tpa <playername> Request to teleport to another player.
/tpahere <playername> Request another player to teleport to you.
/tpaccept Accepts another player's teleportation request.
/tpdeny Denies another player's teleportation request.
/tptoggle Toggles whether you will receive teleport requests.

Chat

Command Purpose
/msg <playername> <msg> Sends a direct message to another player.
/r <msg> Replies to the most recent direct message received.
/ignoreplayer <playername> Ignores chat messages from a specified player.
/unignoreplayer <playername> Makes chat messages viewable from a previously ignored player.
/ignorelist Display a list of people that are on the player's ignore list.
/mail read Displays any mail that the player has in their inbox.
/mail send <playername> <msg> Send a specified player an email.
/mail clear Clears the player's inbox.

Create Images

Creating custom images is a restricted/priveleged ability. Please contact admin for more information.

Command Purpose
/image create <url> <scale> Creates a custom image.
/image delete Deletes a custom image.
